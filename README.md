## Installation Procedure
 * Clone this repo to to your local machine 
 * Make sure you have Mongo runnign on your machine 
 ```sh
 mongo
 ```
 *install your dependencies by running 
 ```sh
 npm install
 ```
 *run the app
 ```sh
 node server.js
```
